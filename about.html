About

The Casual Gamer is an academic game review website that aims to help causal gaming youths and young adults find games to play on PC, Console and Mobile platforms.

Through this website, various web technologies are demonstrated such as HTML, CSS, Javascript and RESTful web APIs. Full website project code can be found on sgguy/gamewebsite GitHub repository.
